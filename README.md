> **CroMM-VSR: Cross-Modal Memory Augmented Visual Speech Recognition**<br>
> Minsu Kim, Joanna Hong, Sejin Park, and Yong Man Ro<br>
> Paper: https://ieeexplore.ieee.org/abstract/document/9566778<br>
